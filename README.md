# slearn

  Machine learning library in Scala -- Built from scratch.


To compile and run:
--------------

  sbt run


To test:
--------------

  sbt test

To run tests in background:

  sbt ~test


Interactive environment:
--------------

  sbt console

  > import slearn.Stats

or:

  cd src/main/scala
  scala
  > import slearn._
